# Day 11 – ATS Resume Optimizer & Resume Generator

## Objective
Learned how to optimize a resume using Claude for Applicant Tracking Systems (ATS).

## What I Did
- Uploaded my existing resume.
- Used a Full Stack Developer Job Description.
- Generated ATS Match Score.
- Reviewed Gap Analysis.
- Optimized my resume without adding fake skills or experience.
- Exported the optimized resume as PDF.

## Files Included
- ATS Analysis Screenshots
- Optimized Resume PDF
- Resume Screenshot

## Key Learnings
- ATS systems scan resumes using keywords.
- Job descriptions should guide resume customization.
- Never add skills or experience that you don't actually have.
- Strong formatting improves ATS readability.
- Gap analysis helps identify areas to learn for future roles.
