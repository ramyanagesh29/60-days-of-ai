# Day 17 – AI Vehicle Cost & Fuel Analysis Dashboard

## Objective

Learned how to transform raw CSV data into an interactive HTML dashboard using Claude AI.

---

## What I Did

- Uploaded a vehicle dataset (CSV)
- Used Claude to analyze fuel data
- Generated an interactive HTML dashboard
- Compared different fuel types
- Visualized KPIs and charts

---

## Dashboard Features

- KPI Cards
- Cost/km Analysis
- CO₂ Analysis
- Maintenance Analysis
- Fuel Comparison
- Age-based Cost Trends
- E85 Score
- Interactive Charts

---

## Key Learnings

- AI can analyze CSV datasets automatically.
- Claude can generate complete HTML dashboards.
- Data visualization improves understanding.
- Prompt Engineering enables reusable analytics workflows.
