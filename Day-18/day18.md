# Day 18 – Brain Dump Action Planner Skill

## Objective

Learned how to build a reusable Claude Custom Skill that transforms unstructured notes into organized project dashboards.

---

## What I Did

- Created a reusable Claude Custom Skill.
- Converted meeting notes into an interactive HTML dashboard.
- Automatically generated summaries, action items, blockers, risks, and open questions.
- Organized messy information into a structured project management view.

---

## Dashboard Features

- Executive Summary
- Key Takeaways
- Action Items Table
- Open Questions
- Risks & Blockers
- Conflict Detection
- Additional Notes
- Interactive Dashboard

---

## Key Learnings

- AI can organize unstructured notes into structured information.
- Custom Skills improve productivity by automating repetitive workflows.
- HTML dashboards provide a clear visual representation of project information.
- Prompt Engineering enables reusable AI-powered project planning.

## Tools Used

- Claude AI
- Claude Custom Skills
- HTML
