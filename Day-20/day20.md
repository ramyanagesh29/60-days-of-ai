# Day 20 – AI Face Puzzle Game

## Objective
Build an interactive Face Puzzle Game using Claude AI.

## Features
- Camera Access
- Capture Face Image
- 3×3, 4×4, 5×5 Difficulty Levels
- Drag and Drop Puzzle
- Touch Support
- Timer
- Move Counter
- Correct Piece Detection
- Win Screen
- Leaderboard using Local Storage
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Canvas API
- MediaDevices API
- Local Storage

## Learning Outcomes
- Learned browser Camera API
- Learned image slicing using Canvas
- Implemented drag-and-drop puzzle mechanics
- Used Local Storage to save scores
- Built a complete browser game with AI assistance
